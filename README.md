# 🚀 Paula Velez Portfolio — v2

> **Currently under active development.**

## 👋 About

This project is the evolution of my previous portfolio, rebuilt from scratch after months of learning, collaborating with digital agencies, and refining my approach to frontend development.

Rather than simply adding new features or visual effects, the goal is to create a portfolio that better reflects how I build digital products—combining product thinking, scalable frontend architecture, motion with purpose, and thoughtful user experiences.

Every section is being reimagined with intention, balancing design, performance, accessibility, and maintainability.

# 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/PaulaVelezz/PaulaVelez.git
```

Go into the project

```bash
cd PAULAVELEZ
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Create a production build

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
